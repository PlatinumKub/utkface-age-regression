# utkface-age-regression
Проект по оценке возраста по фотографиям лиц с помощью моделей глубокого обучения (PyTorch) на датасете UTKFace.

## Описание проекта

- Задача: предсказать возраст человека по изображению его лица (регрессия).
- Датасет: [UTKFace](https://www.kaggle.com/datasets/jangedoo/utkface-new) — более 20k изображений с разметкой возраста.
- Технологии: PyTorch, Python, компьютерное зрение, сверточные нейросети.

## Архитектура и обучение

- Модель: сверточная нейросеть / ResNet (регрессионная голова поверх фич).
- Вход: изображения лиц, приведённые к единому размеру.
- Метрики: MAE / MSE по валидационной выборке.
- Обучение: train/val/test разбиение с фиксированным random seed.
